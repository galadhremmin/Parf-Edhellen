Usage: /persistMethod 
Switches between different persistent modes. There are presently two that you might choose from:

WINDOW
Stores all your configuration variables in the browser window. If you refresh the browser window, your configuration will be reset.

LOCALSTORAGE
Stores all your configuration variables in a browser storage module. This is the option to prefer if you want to be able to refresh the browser window at any time without loosing your configuration. Return to the world you were last connected to by using the /resume command.

Veranda informs you what storage method you're using when you first load the chat.
